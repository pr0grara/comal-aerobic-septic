# Image Attribution

Placeholder images for this site. **All images below are sourced from Wikimedia Commons
under permissive licenses** (CC0, CC BY, CC BY-SA) and verified via `scripts/commons_img.py`.
The previous batch of hand-sourced images (unknown provenance from web image searches) was
replaced on 2026-06-25 because their licensing could not be confirmed.

> These remain **placeholders** (SPEC §2) — real local job photos beat any stock image for a
> site like this. They are here only so the site can launch and rank with legally clean imagery
> while real photos get collected.

CC BY / CC BY-SA **require** attribution (and CC BY-SA requires share-alike). CC0 does not, but
we attribute everything uniformly so the record is complete. Keep this file with the repo.
